# 🎯 AI 图像生成质量评分细则总表

---

## 一、提示词识别度
**核心：模型能否听懂、理解对、执行准**

| 评分细项 | 具体要求 |
| :--- | :--- |
| **文字描述的理解与执行准确性** | - 核心语义无偏差、无歧义、无错解 |
| **要求元素是否完整生成** | - 主体、场景、风格、构图全部出现，无缺失、无简化 |
| **关键词重点内容是否优先突出** | - 高权重元素视觉突出，主次关系清晰，不颠倒、不弱化 |
| **多元素共存稳定性** | - 所有要求元素完整呈现，互不冲突、不丢失、不篡改 |
| **局部编辑指令定位精准度** | - 编辑范围与指令一致，不溢出、不遗漏、不破坏原图 |

---

## 二、功能效果
**核心：功能本身有没有用、能不能达到预期**

| 评分细项 | 具体要求 |
| :--- | :--- |
| **人物、物体、场景结构是否合理** | - 五官、肢体、产品、场景无畸形、无扭曲、无反透视 |
| **细节真实自然度** | - 纹理、发丝、边缘清晰，无伪影、无脏点、无糊化、无错位 |
| **光影统一性与逻辑性** | - 光源方向统一，明暗过渡自然，无突兀、无穿帮 |
| **色彩还原准确度** | - 与参考图/提示词一致，无偏色、无异常变色、无色差 |
| **主体抠取边缘质量** | - 边缘干净，无锯齿、白边、残留，复杂区域完整保留 |
| **去除杂物、水印、瑕疵能力** | - 瑕疵完全清除，修复自然，无补丁、无残留 |
| **合成后真实融合度** | - 与背景光影、透视、质感统一，无拼接、无违和 |
| **局部修改准确性** | - 仅修改目标区域，不扩散、不溢出，衔接自然 |
| **放大、清晰化效果** | - 细节增强，不糊、不锐化过度、不假细节 |

---

## 三、一致性
**核心：产品/模特/场景局部细节还原度是否一致**

### 3.1 产品一致性
| 评分细项 | 具体要求 |
| :--- | :--- |
| **产品形态统一** | 外形、结构、比例不变形 |
| **产品视觉属性统一** | 颜色、材质、纹理统一 |
| **产品细节完整性** | logo、文字、细节无错位/丢失 |
| **产品光影视角统一** | 透视、角度、光影一致，无突变、无偏移 |
| **产品结构完整性** | 部件齐全，无畸变、无错位 |

### 3.2 人物一致性
| 评分细项 | 具体要求 |
| :--- | :--- |
| **人物面部特征统一** | 五官、脸型、发型一致，无变化、无变形 |
| **人物肤质妆容统一** | 肤色、光影、妆容一致，无突变 |
| **人物形体稳定** | 体态、比例、身高一致，无变形 |
| **人物穿搭统一** | 服装、配饰款式、颜色、位置一致 |
| **人物神态气质统一** | 表情、年龄、风格一致，无割裂 |

### 3.3 场景一致性
| 评分细项 | 具体要求 |
| :--- | :--- |
| **场景环境统一** | 环境、空间、背景无增减，完全一致 |
| **场景光影统一** | 光源、强度、色调一致，无突变 |
| **场景氛围统一** | 景深、虚化、色调一致 |
| **构图画幅统一** | 构图、比例稳定，无偏移 |
| **场景整体氛围统一** | 色调、风格、氛围连贯统一 |

---

## 四、画面视觉效果
**核心：从视觉层面评估图片**

### 4.1 构图与版式
| 评分细项 | 具体要求 |
| :--- | :--- |
| **视觉焦点合理** | 重心突出，落点自然，不偏、不飘、不堵 |
| **画面空间舒适度** | 留白适中，透气不拥挤、不松散 |
| **画面视觉平衡** | 均衡稳定，无轻重失衡 |
| **元素规整度** | 分组清晰、间距统一、无杂乱堆砌 |
| **整体比例合理性** | 主体、装饰、文字块比例协调，符合美学逻辑 |
| **透视准确性** | 无拉伸、无反透视、无变形 |
| **网格 / 对齐严谨** | 基线、边距、中轴线统一 |
| **画面整洁度** | 无碎点、无线头、无多余装饰 |

### 4.2 信息层级
| 评分细项 | 具体要求 |
| :--- | :--- |
| **信息主次清晰度** | 核心信息突出，识别、对比最强 |
| **信息逻辑层级** | 主次分明不重复，重点、辅助、装饰层级清晰 |
| **阅读舒适度** | 视觉路径流畅，符合从上到下、从左到右、从大到小 |
| **场景匹配度** | 适配使用场景，优先级符合海报/主图/详情等要求 |

### 4.3 字体与排版
| 评分细项 | 具体要求 |
| :--- | :--- |
| **字体风格契合度** | 气质与主题匹配，字体与画面、品牌调性一致 |
| **字体简洁度** | 数量简洁，不超过2种，无杂乱混搭 |
| **字号逻辑** | 层级清晰，标题 > 副标题 > 正文 > 注释 |
| **字体粗细层次** | 字重对比合理，粗中细层次明确，易区分 |
| **排版舒适度** | 行距字距均匀，不过松、不过紧、不忽大忽小 |
| **文字对齐规范** | 对齐统一，不混用对齐方式，中英文混排协调 |
| **无错字漏字** | 无错字、漏字、多字、乱码、变形 |

### 4.4 色彩与光影（补充）
- 色彩搭配和谐，符合品牌或场景情绪
- 光影塑造立体感，不死板、不扁平
- 高光/阴影过渡自然，无断层、噪点

---

## 五、图片输出规范
**核心：最终成品能否商用、出图**

| 评分细项 | 具体要求 |
| :--- | :--- |
| **图像清晰度** | - 画面清晰，放大无模糊、无噪点、无马赛克 |
| **输出规格合规性** | - 尺寸、比例符合平台规范<br>- 色彩模式符合运用场景 (RGB/CMYK) |
| **商用版权合法性** | - 不抄袭、不盗用、不模仿他人设计<br>- 图片、插画、字体、素材均可商用<br>- 无国旗、国徽、党旗、人民币等违规元素<br>- 无宗教、敏感、暴力、低俗元素<br>- 无侵权肖像、商标、logo |

---

## ✅ 使用说明建议

1. **评分方式**：每项可按 **0~5 分制** 或 **✅/❌ 通过制** 打分。
2. **适用场景**：
   - AI 图像生成模型评测
   - A/B 测试对比
   - Prompt 优化反馈
   - 客户交付验收标准
3. **重点关注**：“一致性”类指标特别适合用于 **电商商品图、虚拟模特、广告素材批量生成场景**。
4. **自动化结合**：可结合自动化脚本 + 人工复核进行批量评估。
